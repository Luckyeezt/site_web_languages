Instruction - INSERT INTO

  • L'instruction INSERT INTO/insert into nous permet de créer une nouvelle ligne (nouvel enregistrement de donnée) dans la base de données.
  • Si vous souhaitez ajouter de nouveaux enregistrements avec INSERT INTO vous devrez utiliser VALUES/values afin d'y ajouter des valeurs.

Comment ajouter des valeurs dans une table TABLE avec l'instruction INSERT INTO ?

  • L'instruction INSERT INTO permet de créer un nouvel enregistrement dans la TABLE.
  • L'instruction TABLE désigne une quelconque TABLE dans la base de données
  • La combinaison des deux instructions CREATE TABLE permet de créer une nouvelle TABLE dans la base de données.


Insertion (INSERT INTO) de valeurs (VALUES) dans la table nommée "celebrities":

/!\ -- Les valeurs doivent être au même emplacement que les noms des colonnes.

INSERT INTO celebrities (nom,age,profession) 
    VALUES("Jordan",25,"Formateur en SQL")
  1. Ajout des instructions INSERT INTO afin d'indiquer que l'on insère des valeurs dans une TABLE.
  2. A la suite, on ajoute le nom de la TABLE que l'on souhaite, ici j'ai choisi "celebrities".
  3. Ensuite, on utiliser l'instruction VALUES en ouvrant les parenthèses afin d'indiquer l'entrée de leurs valeurs respectives.
  4. ("Jordan",25,"Formateur en SQL") sont un paramètre qui contiennent les valeurs.
    • "Jordan" est une valeur de type TEXT qui correspond à la colonne nom.
    • 25 est une valeur de type INT qui correspond à la colonne age.
    • "Formateur en SQL" est une valeur de type TEXT qui correspond à la colonne profession.


Résultat de la TABLE crée aussitôt :


nom age profession
Jordan 25 Formateur en SQL

Comment ajouter plusieurs valeurs dans une table TABLE avec l'instruction INSERT INTO ?



Insertion (INSERT INTO) de plusieurs valeurs (VALUES) dans la table nommée "celebrities":

/!\ -- Les valeurs doivent être au même emplacement que les noms des colonnes.

INSERT INTO celebrities (nom,age,profession) 
    VALUES("Jordan",25,"Formateur en SQL"),
        ("Emma",30,"Développeur Web"),
        ("Alex",28,"Data Scientist");
  1. Ajout des instructions INSERT INTO afin d'indiquer que l'on insère des valeurs dans une TABLE.
  2. A la suite, on ajoute le nom de la TABLE que l'on souhaite, ici j'ai choisi "celebrities".
  3. Ensuite, on utiliser l'instruction VALUES en ouvrant les parenthèses afin d'indiquer l'entrée de leurs valeurs respectives.
  4. ("Jordan",25,"Formateur en SQL"), ("Emma",30,"Développeur Web"), et ("Alex",28,"Data Scientist") sont des paramètres qui contiennent les valeurs pour chaque enregistrement.
  5. /!\ -- Il faut séparer ces paramètres avec des virgules et pour la dernière avec un point virgule ';'.
    • "Jordan", "Emma", et "Alex" sont des valeurs de type TEXT qui correspondent à la colonne nom.
    • 25, 30, et 28 sont des valeurs de type INT qui correspondent à la colonne age.
    • "Formateur en SQL", "Développeur Web", et "Data Scientist" sont des valeurs de type TEXT qui correspondent à la colonne profession.


Résultat de la TABLE crée aussitôt :


nom age profession
Jordan 25 Formateur en SQL
Emma 30 Développeur Web
Alex 28 Data Scientist
