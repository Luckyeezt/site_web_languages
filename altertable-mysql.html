Instruction - ALTER TABLE

  • L'instruction ALTER TABLE/alter table est utilisé pour ajouter,supprimer ou modifier des colonnes dans une table existante.
  • Elle est aussi utilisée pour ajouter ou supprimer différentes contraintes à une TABLE

Ajouter une ou des colonnes à une TABLE avec ALTER TABLE

  • L'instruction ALTER TABLE permet de définir sur quel TABLE nous allons effectuer les modifications.
  • L'instruction ADD COLUMN permet d'ajouter une colonne à la TABLE.
  • Il faut spécifier le nom de la colonne et le type de données (types de données)


Ajout d'une colonne "nationalité" dans la TABLES "celebrities"

ALTER TABLE celebrities
    ADD COLUMN nationalité TEXT;
  1. L'instruction ALTER TABLE indique sur quelle TABLE on va effectuer les modifications
  2. L'instruction ADD COLUMN permet d'ajouter ADD une colonne COLUMN dans la TABLE spécifiée
  3. Une fois l'instruction exécutée, une nouvelle colonne sans données sera ajoutée.


Résultat de la requête ALTER TABLE :


nom age profession nationalité

Ajout d'une colonne "nationalité", "ville_de_naissance", et "twitter" dans la TABLE "celebrities"

ALTER TABLE celebrities
    ADD COLUMN nationalité TEXT,
    ADD COLUMN ville_de_naissance TEXT,
    ADD COLUMN twitter TEXT;
  1. L'instruction ALTER TABLE indique sur quelle TABLE on va effectuer les modifications
  2. L'instruction ADD COLUMN permet d'ajouter ADD une colonne COLUMN dans la TABLE spécifiée
  3. Une fois l'instruction exécutée, trois nouvelles colonnes sans données seront ajoutées.


Résultat de la requête ALTER TABLE :


nom age profession nationalité ville_de_naissance twitter
