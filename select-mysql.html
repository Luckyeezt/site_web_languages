Instruction - SELECT

  • L'instruction SELECT/select nous permet de récupérer des valeurs à partir d'une base de données

Comment récupérer des données depuis une table avec l'instruction SELECT en MySQL ?

  • L'instruction SELECT permet de récupérer des données depuis une table dans une base de données MySQL.
  • L'instruction FROM désigne la table spécifique à partir de laquelle les données seront extraites.
  • Les colonnes spécifiques à récupérer sont spécifiées après l'instruction SELECT.


Récupération de données depuis la table nommée "celebrities" :

/!\ -- Les colonnes spécifiées doivent correspondre aux noms de colonnes de la table.

SELECT nom, age, profession 
    FROM celebrities;
  1. L'instruction SELECT indique les colonnes spécifiques à récupérer, ici "nom", "age" et "profession".
  2. L'instruction FROM est utilisée pour spécifier la table "celebrities" à partir de laquelle les données seront récupérées.
  3. Une fois l'instruction exécutée, les données de toutes les colonnes spécifiées seront récupérées pour chaque enregistrement dans la table.


Résultat de la requête SELECT :


nom age profession
Jordan 25 Formateur en SQL
Emma 30 Développeur Web
Alex 28 Data Scientist
