Introduction au SQL

  • Une base de données est un ensemble de données qui ont été stockées sur un support informatique, organisées et structurées de manière à pouvoir modifier et consulter leurs données facilement.
  • Dans SQL, le langage de requête structuré, les données sont manipulées à l'aide de différentes instructions telles que SELECT, INSERT, UPDATE, DELETE, etc.
  • Dans un système de gestion de base de données (SGBD) comme mySQL, les données sont organisées dans des tables. Chaque table est un ensemble de données structurées sous forme de lignes et de colonnes.
  • Par exemple, nous pourrions avoir une TABLE appelée "Employés" qui contient des informations sur les employés d'une entreprise :
TABLE : Employés
ID Nom Âge Poste
1 John Doe 35 Ingénieur logiciel
2 Jane Smith 28 Analyste financier
3 Michael Johnson 42 Directeur des ventes

Dans cette table, chaque ligne représente un employé avec des informations telles que son identifiant (ID), son nom, son âge et son poste.

  • Les règles de nommage des noms de la TABLE :

  • Utilisez le snake_case si vous prévoyez un nom avec plusieurs mots : user_name_id
  • Ne mettez pas d'espaces pour séparer les mots de votre TABLE / colonne
  • Ne pas utiliser de mots réservés aux propriétés sql pour les noms de vos colonnes : date => date_inscription
