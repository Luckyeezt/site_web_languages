Instruction - CREATE

  • L'instruction CREATE/create nous permet de créer une nouvelle TABLE dans la base de données ou une nouvelle DATABASE. Il faut utiliser l'instruction CREATE afin d'avoir une table créer à partir de zéro.
  • Si nous souhaitons créer une nouvelle TABLE, nous devrons inscrire directement les champs ainsi que leurs types que cette TABLE contiendra

Comment créer une nouvelle TABLE avec l'instruction CREATE ?

  • L'instruction CREATE permet de créer un nouveau "item".
  • L'instruction TABLE désigne une quelconque TABLE dans la base de données
  • La combinaison des deux instructions CREATE TABLE permet de créer une nouvelle TABLE dans la base de données.


Création d'une TABLE : 

CREATE TABLE celebrities (
    nom TEXT,
    age INT,
    profession TEXT
);
  1. Ajout des instructions CREATE TABLE afin d'indiquer que l'on créer une TABLE
  2. A la suite, on ajoute le nom de la TABLE que l'on souhaite, ici j'ai choisi "celebrities"
  3. Ensuite, on ouvre la parenthèses afin d'indiquer les noms des colonnes et d'indiquer leurs types de données.
  4. Ici, j'ai créer 3 colonnes nommées "nom","age","profession", j'ai indiquer ensuite leurs types de données.
  5. Pour créer plusieurs noms de tables, nous devons les séparer avec des virgules en fin de ligne.
  6. Nous fermons ensuite la parenthèses avec un point-virgule ";" afin de créer la table.


Résultat de la TABLE crée aussitôt :


nom age profession

Comment créer une nouvelle DATABASE avec l'instruction CREATE ?

  • L'instruction CREATE permet de créer un nouveau "item".
  • L'instruction DATABASE désigne une quelconque base de données SQL.
  • La combinaison des deux instructions CREATE DATABASE permet de créer une nouvelle DATABASE SQL.


Création d'une TABLE : 

CREATE DATABASE jeux_videos
  1. Ajout des instructions CREATE DATABASE afin d'indiquer que l'on créer une DATABASE.
  2. A la suite, on ajoute le nom de la DATABASE que l'on souhaite, ici j'ai choisi "jeux_videos".
